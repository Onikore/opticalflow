# Паспорт физических пределов: STM32H743 + ATK-OV7725 MD v1.1

Пределы исходной связки железа, посчитанные из наших измерений (шумовой пол, потолок
пирамиды, GSD-вымывание, rolling shutter) и **реальных спеков камеры**. Формулы даны —
пересчитываются под любой замер.

## Камера (из спецификации ALIENTEK)

| параметр | значение | следствие |
|---|---|---|
| Сенсор | OV7725, 1/4", 656×488, пиксель **6 мкм** | крупный пиксель → хорошая светосила |
| Объектив | 3.6 мм, **F2.0**, диагональный FOV **78°** | см. фокус ниже |
| Режимы | **VGA 640×480@60**, **QVGA 320×240@120** | два опорных режима |
| Чувствительность | 3.0 В/(люкс·с), SNR 50 дБ, DR 60 дБ | короткие экспозиции реальны |
| FIFO | AL422B, **384 КБ** | ограничивает формат VGA (ниже) |
| Rolling shutter | прогрессивный, RS | измерено: минорен для потока |

**Фокус в пикселях (native VGA):** pinhole 3.6мм/6мкм = **600 px**; из диагонального
FOV 78° = **494 px** (расхождение — дисторсия широкоугольника: краевой FOV шире
пинхольного). Рабочая оценка **f_native ≈ 500–600 px** (HFOV ≈ 56–66°).
⚠️ На железе замерить одной фотографией линейки с известной дистанции — все числа
ниже линейны по f.

**Ограничение FIFO (важно!):** AL422B = 393 216 Б.
- QVGA YUV422 = 153.6 КБ — влезает ✓ (@120fps)
- **VGA YUV422 = 614 КБ — НЕ ВЛЕЗАЕТ** ✗
- VGA RawRGB (Bayer, 8 bpp) = 307 КБ — влезает ✓
→ VGA@60 доступна **только в RAW Bayer**; яркость брать из G-пикселей (шахматная
субдискретизация — эффективное разрешение канала света чуть ниже, для потока приемлемо).

## Два канала из одного сенсора (цифровой кроп)

- **A: wide** — QVGA@120fps → даунскейл 64×64: f≈55 px, FOV≈60°, dt=8.3 мс
- **C: crop («телевик»)** — VGA@60fps (RAW) → нативный центр-кроп 64×64:
  **f≈550 px, FOV≈6.6°**, dt=16.7 мс

## Таблица пределов

| Предел | Формула | A: wide (f=55, 120fps) | C: crop (f=550, 60fps) |
|---|---|---|---|
| Потолок потока (пирамида 9.5 px) | `9.5/(f·dt)` | 20.6 рад/с (~1180°/с) | 1.03 рад/с (**59°/с**) → нужен гиро-гейт/центрирование |
| Макс. скорость | `rate·h` | 20·h м/с | 1.0·h м/с |
| Шум скорости (1 с усредн., σ_px≈0.08) | `σ·h/(f·dt·√fps)` | **1.6 см/с × h** (h=10: 16 см/с; h=30: 47) | **0.11 см/с × h** (h=100: 11 см/с) |
| Невидимый крип (порог 0.1 px/кадр) | `0.1·h/(f·dt)` | 0.22·h м/с (!) | 0.010·h м/с |
| GSD (вымывание текстуры) | `h/f` | 1.8 см × h (h=30 → 55 см/px: каша) | 0.18 см × h (h=100 → 18 см/px: ок) |
| Rolling shutter (изм.) | ∝ f·Tread·Δω | 0.07–0.5 px, ничтожен | до ~2.7 px на угловом ускорении → гейт по Δω |

## Дальномер и IMU

- **VL53L1X**: 4 м в тени, **1–2 м на ярком солнце**, 50 Гц. Единственный ЖЁСТКИЙ
  потолок метрического масштаба. Выше — масштаб от баро через EKF (наш выход в
  радианах это позволяет; доказано на реальном полёте до 30 м).
- **ICM-42688**: шум/биас гиро → ошибка де-ротации; мульти-IMU даёт √N + медиана-
  голосование против сбоя (см. EXPERIMENTS). Оси/знак калибровать на столе
  (SensorConfig, знаковый тест: чистое вращение ⇒ v≈0).

## Вычисления и свет (H743 @480 МГц)

- Поток: <1 мс/канал с `__USADA8` — не лимит. Оба кропа из одного кадра бесплатно.
- **Ботлнек — чтение FIFO битбэнгом**: тактовать весь кадр: QVGA ≈ 2–3 мс (ок @120fps),
  VGA ≈ 9–10 мс (впритык @60fps) → **TIM+DMA обязателен для VGA**.
- Свет: 6 мкм + F2.0 + 3 В/(люкс·с) → экспозиция 8 мс (@120fps): улица/пасмурно с
  запасом, тусклое помещение маргинально. **AEC/AGC выключить по SCCB** (обязательное).

## Неочевидные выводы

1. **Fps — двусторонний размен**: высокий fps даёт огромный потолок динамики, но
   портит чувствительность к крипу (за 8 мс дрейф смещает суб-0.1 px → слепота +
   S-кривая занижения). Правильно — **адаптивный dt**: на зависании копить кадры
   (пропускать), на манёвре считать каждый.
2. **Крип-слепота — главный предел удержания** wide-канала: на h=10 не видит снос до
   ~2 м/с. Лечится только фокусом → crop-канал (0.1 м/с на той же высоте).
3. **Эшелоны исходного железа (без замены компонентов, только прошивкой):**
   - **0.3–4 м** — полная автономия (VL53 жив): удержание ±доли метра, шум 2–5 см/с
   - **4–30 м** — камера отличная (баг-2: 100% валидность на 30 м), масштаб от баро
   - **30–100 м** — только crop-канал + гиро-центрирование + баро; шум 3–11 см/с;
     нужен дальнобойный лидар для terrain, иначе точность масштаба = баро
4. Rolling shutter и вычислительный бюджет — НЕ ограничения (измерено/посчитано).
   Реальные рычаги дальше: дальномер подальше, объектив поуже (или FullHD-сенсор
   с кропами — см. EXPERIMENTS, эшелон до 200 м), фиксированная экспозиция.
